# Taxman – Web App

Next.js app for the Taxman AI tax copilot. See the [root README](../README.md) for setup, env vars, and features.

## Development

```bash
npm install
cp .env.example .env   # Add OPENAI_API_KEY, etc.
npm run dev
```

Open [http://localhost:5001](http://localhost:5001).

## Scripts

- `npm run dev` – Start dev server (port 5001)
- `npm run build` – Production build
- `npm run start` – Run production server
- `npm run lint` – Run ESLint
